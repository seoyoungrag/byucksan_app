GetBodyValue

현재 편집 중인 문서의 HTML body정보(<body>~</body>)를 얻어 옵니다.

Syntax

    HRESULT GetBodyValue();

Parameters

없음

Return Values

body 정보를 문자열(XHTML 문법)로 반환합니다.

Remarks

Method는 Main class 호출 후 에디터가 로드된 다음 실행해야 정상적으로 실행됩니다.

Sample Codes

JavaScript

var object = new NamoSE(“test”);
object.editorStart();
var info = object.GetBodyValue();
alert(info);
