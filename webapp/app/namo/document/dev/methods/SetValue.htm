SetValue

현재 편집 중인 문서의 내용 전체를 HTML 형식으로 설정합니다.

Syntax

    HRESULT SetValue(
          BSTR newVal
    );

Parameters

newVal

[in] 변환할 문서의 내용을 입력합니다.

Return Values

없음

Remarks

Method는 Main class 호출 후 에디터가 로드된 다음 실행해야 정상적으로 실행됩니다.

Sample Codes

JavaScript

var object = new NamoSE(“test”);
object.editorStart();
object.SetValue(“<html><head><title>test</title></head><body><table><tr>
<td>test 문서 입니다.</td></tr></table></body></html>”);
