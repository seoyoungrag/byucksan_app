Method

Event

 

 

SetBodyStyle v1.1 or higher

현재 편집 중인 문서의 HTML body 태그의 스타일 속성을 설정합니다.

Syntax

    HRESULT SetBodyStyle(
          BSTR styleName,
          BSTR value
    );

Parameters

styleName

[in] 지정할 스타일의 속성명을 입력합니다.

value

[in] 지정할 스타일의 속성값을 입력합니다.

Return Values

없음

Remarks

크로스에디터의 Main class 호출 후 에디터가 로드된 다음 실행해야 정상적으로 실행됩니다.

Sample Codes

JavaScript

var object = new NamoSE(“test”);
object.editorStart();
object.SetBodyStyle('background-color','#FF0000');
