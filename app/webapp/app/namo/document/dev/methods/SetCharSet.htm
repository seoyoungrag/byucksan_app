SetCharSet v1.1 or higher

현재 편집 중인 문서의 HTML meta 태그의 charset를 변경합니다.

Syntax

    HRESULT SetCharSet(
          BSTR newVal
    );

Parameters

newVal

[in] 수정할 charset을 입력 합니다.

Return Values

없음

Remarks

크로스에디터의 Main class 호출 후 에디터가 로드된 다음 실행해야 정상적으로 실행됩니다.

Sample Codes

JavaScript

var object = new NamoSE(“test”);
object.editorStart();
object.SetCharSet(“euc-jp”);
